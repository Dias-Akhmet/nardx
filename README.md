### Product Description

#### What is NardX?
NardX is a premium, web-based implementation of Long Backgammon (Nardy) built with React, TypeScript, and Tailwind CSS. It transforms a traditional board game into a modern, highly engaging, and competitive digital platform.

#### Who is it for?
* **Board Game Enthusiasts:** Players looking for a flawless, modern, and mathematically accurate way to play Long Backgammon online or locally.
* **Competitive Strategy Players:** Users who want to improve their skills, analyze their mistakes, and solve tactical puzzles.
* **Modern Audience:** Players who appreciate high-end minimalist aesthetics, responsive micro-animations, and dynamic audio-visual themes.

#### Why is it valuable? (Core Features)
* **Flawless Long Backgammon Logic:** Strict enforcement of traditional rules, including the Head rule, block checks, automated turn lifecycles, and smooth checker drop animations.
* **Smart AI Companion:** Includes a fully responsive automated bot for solo practice and a Post-Game Review system that analyzes matches, highlighting blunders, mistakes, and brilliant moves.
* **Gamification and Engagement:** Features interactive daily quests and tactical puzzles to maximize user retention.
* **Monetization-Ready Architecture:** Built-in marketplace featuring lockable premium skin packs (such as Neon Cyberpunk and Royal Marble) and an unlocked Pixel Retro theme equipped with unique 8-bit sound effects.
* **Sleek, High-End UI:** A minimalist, dark-themed dashboard layout optimized for high performance and immersive audio feedback.
